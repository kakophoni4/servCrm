# Экраны по фазам

## Фаза 1 — MVP (2–3 нед)

| # | Экран | Роли |
|---|-------|------|
| 1 | Логин | все |
| 2 | Заявки (список + фильтры) | дисп+, админ+ |
| 3 | Создание / редактирование заявки | дисп+, админ+ |
| 4 | Карточка заявки (статус, мастер, оплаты, история клиента) | по правам |
