# Формулы денег, ЗП, отчётов

Зафиксировать 1:1 с Excel заказчика (когда Excel будет загружен в `docs/DATA/`).

## Базовые (заявка)

```
work_sum      = paid_by_client - parts_cost
master_pct    = category_by(work_sum).percent
master_salary = work_sum * master_pct
to_company    = work_sum - master_salary
```

## Правило «Готов»

Если `paid > 500` → обязательны подтверждающие документы.
Сумма сохраняется; без чеков статус `DONE` недоступен.
На MVP — предупреждение; жёсткая блокировка с фазы 2.

## Прогноз оборота (отчёт, не MVP)

```
forecast = avg_check_month * avg_closed_per_day * days_in_month
```

Уточнить усреднение у заказчика (см. OPEN-QUESTIONS).

## Отчёты (фаза 3) — черновик метрик

### Закрытые заявки

- сумма расчётов за период (по умолчанию текущий месяц)
- прогноз оборот
- % претензий
- закрытые / наши / партнёрские
- чистая сумма / ср. чек сдачи / ср. чек ЗП / ср. чек общий
- цена заявки = сумма рекламы / количество заявок

### Отмены

- по вине мастера / администратора
- по нашим / партнёрским

### Реклама

- остаток листовок / визиток
- кол-во объявлений Авито / промоутеров / заявок
- KPI листовки = (листовки + визитки) / заявки по листовкам
- KPI Авито = заявки / объявления

### Оборот мастеров

мастер, оборот, 4% от оборота, кол-во заявок, открытых СД,
чистый ср. чек, общий ср. чек, запчасти, микра (<4к), зарплата

### Претензии

все поля формы в удобочитаемом виде

### Касса

город / приход общий / приход с заявок (−ЗП) / прочий приход /
расход промоутеров / инкасс / ЗП мастерам / объявления /
общий расход / остаток
